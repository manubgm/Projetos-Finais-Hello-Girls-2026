# Projetos-Finais-Hello-Girls-2026
Projetos realizados pelas alunas do curso "Introdução a HTML e CSS" ministrado pelo Projeto de Extensão "Hello, Girls!" da UNESP Bauru.
O projeto tem como objetivo incentivar o ingresso de mulheres na área da tecnologia, e este repositório possuí função de reunir e prestigiar o trabalho feito pelas estudantes.
